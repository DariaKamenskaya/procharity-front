# Как сбилдить проект

1. Установить node
   <https://nodejs.org/en/>

2. установить yarn

{

`````bash
npm install -g yarn
````

}

3. Перейти в корень проекта в терминале и установить зависимости

{
```bash
yarn install
`````

} 4. Запустить команду билд
{

```bash
yarn build
```

}
В корне проекта появится папка build
